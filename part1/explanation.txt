This simple Mess3 mixture environment acts as a mechanistic sandbox for understanding Language Models, specifically evaluating how neural sequence models handle non-ergodic environments. A standard text dataset generated by humans is fundamentally a mixture of vastly different underlying generators (e.g., Python code, French literature, Reddit comments), each with distinct stationary distributions and transition dynamics, but where the "topic" remains locally consistent within a single context window.

By training a tiny causal Transformer on this known, fixed mixture of three exact Hidden Markov Models (the Mess3 processes without intra-sequence component switching), we create a simplified analog of in-context learning and induction heads. The model must learn not only the local transition probabilities to accurately predict the next token (standard language modeling), but dynamically infer *which underlying generator* it is currently observing based on early tokens inside the context window. This mimics the fundamental problem an LLM solves: observing a prompt, implicitly narrowing down the active hypothesis space of the generator distribution, and conditioning subsequent predictions on that belief state. Since we can compute the exact Bayesian belief state iteratively via the predictive vectors, we have access to "ground truth" induction tracking that we can directly compare against the Transformer's learned hidden geometry.
